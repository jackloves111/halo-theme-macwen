## 使用说明
1.下载release压缩包或者直接下载源代码，在halo后台上传启用即可  
2.搜索功能配合官方搜索插件 [
plugin-search-widget](https://github.com/halo-sigs/plugin-search-widget)使用  
3.评论功能配合官方插件[
plugin-comment-widget](https://github.com/halo-sigs/plugin-comment-widget)使用  
4.启用后请先把设置中该填的都填一下
